# SkinILLPILL

AI-powered skin lesion classification web application built using deep learning and deployed with Vercel.

## Overview

SkinForVercel is a medical image classification project focused on detecting and classifying skin lesions using deep learning models trained on datasets such as HAM10000 and ISIC.

The project combines:

* Deep Learning
* Computer Vision
* Medical Image Analysis
* Web Deployment

to create an accessible web-based skin lesion prediction system.

---

## Features

* Upload skin lesion images
* AI-based classification and prediction
* Deep learning inference pipeline
* Web deployment using Vercel
* Responsive frontend interface
* Fast image prediction workflow

## Datasets Used

* HAM10000
* ISIC Dataset

These datasets contain dermoscopic images used for training skin lesion classification models.

---

## Model Capabilities

The model is trained to identify multiple categories of skin lesions and assist in automated image-based diagnosis workflows.

Techniques used include:

* Data Augmentation
* Transfer Learning
* Ensemble Learning
* Cross Validation
* Confidence-based Prediction

## Installation

Clone the repository:

```bash
git clone https://github.com/ArunB777/skinforvercel.git
```

Move into the project folder:

```bash
cd skinforvercel
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
